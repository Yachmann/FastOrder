# FastOrder 🍔🚚

FastOrder é um sistema desenvolvido em **Python** para auxiliar restaurantes e pequenos comércios no **gerenciamento de pedidos, clientes, entregadores e pagamentos**, facilitando o controle das entregas e do fluxo de pedidos.

O projeto está sendo desenvolvido para a  **ExpoTech**, seguindo boas práticas de organização, versionamento e entrega incremental.

---

## 📌 Objetivo do Projeto

O objetivo do FastOrder é oferecer uma solução simples e eficiente para:

- Criar e gerenciar pedidos
- Controlar clientes, produtos e entregadores
- Acompanhar o status dos pedidos em tempo real
- Calcular automaticamente o valor total do pedido
- Registrar histórico de pedidos e entregas

---

## 🧾 Escopo do Sistema

O sistema contempla as seguintes funcionalidades:

- Gestão de clientes  
- Cadastro e gerenciamento de produtos  
- Criação e gerenciamento de pedidos  
- Controle de pagamento (dinheiro, cartão ou online)  
- Gestão de entregadores e funcionários  
- Atualização de status do pedido (preparando, saiu para entrega, entregue)  
- Cálculo automático do valor total do pedido  
- Histórico de pedidos
